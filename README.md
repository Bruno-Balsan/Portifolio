# Portfólio Pessoal

Este repositório contém meu portfólio profissional, desenvolvido para apresentar minha formação, experiência prática, cursos e certificados na área de Tecnologia.

Atualmente atuo como **estagiário de TI no 1º Cartório de Registro Civil de Osasco**, onde aplico conhecimentos técnicos no suporte e na organização de recursos tecnológicos, ao mesmo tempo em que sigo em constante aprendizado.


## Tecnologias Utilizadas

- HTML  
- CSS  
- Git  
- GitHub  


## Estrutura do Projeto

- `index.html` — página principal do portfólio  
- `style.css` — estilização do site  
- `certificados/` — pasta contendo os certificados em PDF  


## Cursos e Certificados

O portfólio reúne certificados de cursos voltados ao desenvolvimento técnico e ao raciocínio lógico, entre eles:

- Comunicação e Mobilização com Inteligência Artificial  
- Linguagem de Programação Python e Raciocínio Lógico  
- LGPD  
- Desenvolvimento de Site Simples  

Os certificados estão organizados na pasta `certificados` e acessíveis diretamente pelo site.
